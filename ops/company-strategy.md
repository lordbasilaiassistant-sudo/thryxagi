### Decision Framework
When an agent faces a choice, ask:
1. Does this generate revenue? → Priority
2. Does this increase visibility? → Do it
3. Does this cost ETH? → Only if ROI > 2x within 1 week
4. Is it free? → Always do it
5. Does it risk our reputation? → Skip it
6. Is it reversible? → Try it
7. Does it compound? → Prioritize over one-off actions

### Edge Cases to Handle
- Platform down → Try next platform, report issue
- Wallet limit hit → Rotate to next wallet, update wallet-rotation.md
- Tweet rate limited → Wait 15min, try again
- Browser blocked → Report to VP, try API alternative
- Duplicate ticker → Always check deployed-tokens.md first
- Fee claim fails → Log the failure, try next token
- Token already exists on platform → Skip, deploy different name
- No ETH for gas → Report to Ledger/Thryx immediately
- Agent stuck → Report blocker and shut down (don't idle)
